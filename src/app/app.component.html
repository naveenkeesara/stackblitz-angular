Angular Interview Challenge — User Directory

Read all instructions carefully before writing any code. You have 30 minutes.

What is already built for you

  1. UserService — already has 8 users. Use getUsers() to fetch them. Do NOT modify it.
  2. AppModule — ReactiveFormsModule is already imported for you.
  3. Global styles are ready — focus on logic, not CSS.

Your Tasks

  1. Task 1 — Service & Display
    Create a UserListComponent. Inject UserService and display all users in a list. Show each user's name, email, department, and status.
  2. Task 2 — Parent-Child Communication
    Create a UserCardComponent. Pass each user from the list to the card using @Input(). Add a Delete button inside the card. When clicked, use @Output() to notify the parent to remove that user from the list.
  3. Task 3 — Search Filter (RxJS)
    Add a text input at the top of UserListComponent. Filter the displayed users as the candidate types. Bonus: Use debounceTime(300) so the filter only runs after the user stops typing for 300ms.
  4. Task 4 — Directives & Styling
    Use *ngIf to show a "No users found" message when the list is empty.
    Use ngClass to highlight the card in green if the user's status is Active.
  5. Task 5 — Lifecycle Hook
    Inside UserCardComponent, implement ngOnDestroy. Log this message to the console when a card is destroyed: User [name] removed from the list

User Model (for reference)

Each user object has these fields:

  1. id — number
  2. name — string
  3. email — string
  4. department — string
  5. status — 'Active' or 'Inactive'
