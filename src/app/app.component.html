🌟 Angular Interview Challenge

User Directory — Angular 18 + RxJS

⏰ You have 30 minutes

What is already built for you

  1. UserService with 8 users — call getUsers(). Do NOT modify it.
  2. CommonModule and ReactiveFormsModule are already imported in AppComponent — import them in your own components too.
  3. Global styles are ready — focus on logic, not CSS.

Important: Standalone Components

This project uses Angular 18 standalone components. There is no app.module.ts. Every component you create must have standalone: true in its decorator and must import its own dependencies.

Task 1 — Service & Display

  1. Create UserListComponent as a standalone component
  2. Inject UserService and call getUsers()
  3. Display all users — show name, email, department, status
  4. Import UserListComponent in app.component.ts
  5. Add <app-user-list> in app.component.html

Task 2 — Parent → Child Communication

  1. Create UserCardComponent as a standalone component
  2. Accept a user via @Input()
  3. Display the user details inside the card
  4. Add a Delete button
  5. Emit the user id to the parent via @Output() on click
  6. Parent removes that user from the list

Task 3 — Search Filter (RxJS)

  1. Add a text input at the top of UserListComponent
  2. Use FormControl and filter users as you type
  3. Bonus: Wrap with debounceTime(300) so filter only runs after typing stops for 300ms

Task 4 — Directives & Styling

  1. Show "No users found" message when filtered list is empty — use @if (new syntax) or *ngIf
  2. Add class="active-card" when user.status === 'Active' — style it with a green left border or green text

Task 5 — Lifecycle Hook

  1. Implement ngOnDestroy in UserCardComponent
  2. Log to console when a card is destroyed:
    User [name] removed from the list
